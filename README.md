# 🌌 HoloScan.js

O **HoloScan.js** é um componente interativo desenvolvido em JavaScript puro, HTML5 e CSS3, projetado para criar um efeito visual de scanner holográfico futurista a partir de sequências de imagens. 

Ideal para interfaces de ficção científica (Sci-Fi UI), portfólios modernos ou páginas de transição que buscam alto impacto visual e interatividade fluida baseada no movimento do usuário.

---

## 🚀 Funcionalidades

*   **Efeito Holográfico Avançado:** Camadas de varredura visual que simulam um scanner de ficção científica real.
*   **Sequência de Imagens de Alta Performance:** Renderização otimizada utilizando imagens no formato `.webp`, garantindo carregamento ultrarrápido e transições suaves.
*   **Design Responsivo:** Interface adaptável que se ajusta perfeitamente a diferentes tamanhos de tela e dispositivos.
*   **Estilização Futurista:** Uso de efeitos de brilho (glow), desfoque de fundo (backdrop-filter) e paleta de cores neon.
