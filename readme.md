# HoloScan 🚀

[Português](#português) | [English](#english)

---

## Português

Componente web interativo e responsivo projetado para transformar qualquer imagem ou foto em um holograma futurista dinâmico. Ao substituir o arquivo de mídia, o componente renderiza um efeito visual de varredura (*scanning*), projetando a nova imagem na tela com uma estética sci-fi imersiva. Ideal para landing pages, portfólios e interfaces modernas que buscam um forte impacto visual.

### 📁 Como Substituir a Imagem

Para que o efeito de escaneamento e a projeção do holograma funcionem perfeitamente, siga os requisitos abaixo ao trocar de imagem:

1. **Formato Obrigatório:** A nova imagem deve estar obrigatoriamente no formato **`.webp`** (garantindo melhor performance, transparência e carregamento rápido).
2. **Localização:** O arquivo deve ser colocado exclusivamente dentro da pasta **`/assets`** do projeto.

### 🛠️ Tecnologias Utilizadas

* HTML5 / CSS3 (Efeitos de iluminação, animação de varredura e estética Sci-Fi)
* JavaScript

---

## English

An interactive and responsive web component designed to transform any image or photo into a dynamic, futuristic hologram. When a media file is replaced, the component renders a visual scanning effect, projecting the new image onto the screen with an immersive, sci-fi aesthetic. Perfect for landing pages, portfolios, and modern web interfaces looking to create a strong visual impact.

### 📁 How to Replace the Image

To ensure the scanning effect and hologram projection work flawlessly, make sure to follow these media requirements when swapping images:

1. **Required Format:** The new image must be in **`.webp`** format (ensuring optimal performance, transparency, and fast loading speeds).
2. **Location:** The file must be placed exclusively inside the project's **`/assets`** folder.

### 🛠️ Technologies

* HTML5 / CSS3 (Lighting effects, scanlines, and Sci-Fi animations)
* JavaScript
